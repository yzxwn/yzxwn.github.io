#### 简介
* 使用JavaScript和React编写原生移动应用
* 开发特点：
    * 一次学习，随处编写
    * 混合开发
    * 高效的 UI 开发
    * 高效的 UI 调试
    * 学习门槛低、开发难度低
    * 开发软硬件要求低
    * 使用 React Native 开发的代价：内存消耗略大、运行速度略慢、安装包比原生代码安装包大

#### 初始化
完整原生环境
```
react-native init myProject //新建项目
运行模拟器或连接真机
react-native run-android    //启动项目
```
简易沙箱环境
```
create-react-native-app AwesomeProject
cd AwesomeProject
npm start
手机端expo扫码
```
