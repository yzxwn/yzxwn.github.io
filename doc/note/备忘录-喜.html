混合app开发的经验（react-native） 
            APP的三种开发模式：
                Native App（原生APP）：应用的性能好，适配起来相对容易（无法跨平台、升级麻烦、Store的审核）
                Web App（把手机当做一个浏览器）：Android使用WebView，iOS使用UIWebView，开发成本大大降低（页面访问速度慢、操作体验差）
                Hybrid App（混合APP）：综合了Web App的开发速度和Native App的高性能体验
